# デモ進行用ランブック

## ゴール
- 農家・学生・運営がどのように価値交換するかを7分以内で体験説明。
- 実装済みのUXを中心に、モック化している領域と今後の展開を明示。

## 事前準備
- アプリを起動し、Firebase匿名ログインでサインイン（`AuthGateScreen`）。
- 必要に応じて `AppState.signOut()` を実行して初期状態に戻す。
- Google Maps表示用APIキーを設定済みか確認（未設定の場合はスクリーンショットで代替説明）。
- スクリーン録画またはキャプチャツールを用意。

## デモシナリオ（約7分）
1. **オンボーディング（1分）**
   - ペルソナカード（農家/学生/運営）を紹介。
   - 「ゲストログインで体験開始」を実行しスナックバー通知を確認。

2. **募集検索とフィルタ（1.5分）**
   - `HomeScreen` で検索ボックス・タグフィルタを操作。
   - `ApprovalBanner` 未表示の初期状態を確認。
   - 募集カードタップでフェード＋スライド遷移を解説。

3. **募集詳細〜応募（1.5分）**
   - 地図プレビュー（`FarmMapPreview`）で位置を確認。
   - 応募ボタン→ローディングダイアログ→応募完了Snackbar。
   - 1秒待機後の承認通知（チャット＆ホームのApprovalBanner）を紹介。

4. **チャット・チェックイン・フォトログ（1.5分）**
   - `ChatScreen` で運営ボットとの会話履歴を確認、自身でメッセージ送信。
   - 詳細画面に戻り、チェックイン→ローディング→完了Snackbar。
   - フォトログ追加（ボトムシート→コメント入力→疑似写真表示）を実演。

5. **マイル可視化と予約（1分）**
   - `MilesScreen` の折れ線グラフと履歴カードを説明。
   - `ReservationScreen` でTableCalendar操作→ローディング→予約完了Snackbar。

6. **マイページと将来展望（1分）**
   - `ProfileScreen` → チュートリアル（`TutorialScreen`）で導線再確認。
   - 「省略機能と今後のアップデート」ダイアログで拡張計画を共有。

## メッセージング要点
- **安心感**：柔らかいアニメーション、ローディング演出、スナックバー通知で信頼性を表現。
- **努力の見える化**：マイルグラフとフォトログで成果をビジュアル化。
- **省略機能の説明**：決済、本人確認、実通知はモックで、将来方針を口頭＋ダイアログで補足。
